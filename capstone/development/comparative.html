Capstone Comparative Analysis

Atelier Heschung

The art style is very minimalist but this is the exact style I want my project to have. There is very little user interaction but there is enough to keep the user engaged. With the use of keyboard interactions the project feels slightly like a game. The user experience is very casual but still informative at the same time.

Violence Conjugale

The main point of focus for this interactive website is how the designers use a phone interface to create interaction. I want my project to revolve around a conversation/story and I think this is an interesting way to do it. There is a lot of player interaction and the user is allowed to make choices to manipulate the story, it feels like a visual novel.
