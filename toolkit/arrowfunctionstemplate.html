Arrow Function Example

This function is called greet, when it is called it returns hello world in the console. I came up with this function for this assignment.

Named Function

            function greet() {
                return 'hello world';
            }

Function Expression

            const greet = function(){
                return 'hello world';
            }

Arrow Function

            const greet = () => 'hello world';
